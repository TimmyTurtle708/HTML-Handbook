🧠 心理建設

你學的 HTML 在這裡為什麼不能用？

Email 的對象是郵件客戶端，不是瀏覽器。Outlook 改用 Word 渲染，CSS 支援停留在非常原始的程度。

Outlook 為什麼這麼難搞？

Outlook 2007 後用 Word 當渲染引擎，導致大量 CSS 失效。企業用戶還在用舊版，所以必須為最差的環境設計。

「我明明沒改什麼，怎麼就壞了？」

同樣的 code 在不同地方看起來完全不同。Gmail 網頁版可能正常，但 Outlook 客户端卻跑版。

GitHub 連線設定
在 GitHub → Settings → Developer settings → Personal access tokens 產生，勾選 repo 權限即可。Token 只存在你自己的瀏覽器裡，不會上傳到任何地方。
